# Rider Tombs

Creatures be stirring in the depths of Skyrim's tombs. Long forgotten, these apocalyptic riders await their chance to rise again. The weapons of these riders must be removed from their tombs, apocalypses tend to be bad for the sweet roll business. Slay these goulish souls and protect Skyrim's sweet roll supply.

Rider Tombs adds four new dungeons to Skyrim. The quest for the dungeons will start when any one of the dungeons are entered.

Four total exterior cells are modified, one for each dungeon: (-13,-15), (47,-30), (22,25), (-8,10). No other base game modifications are made.
